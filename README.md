You'll need to provide the following in the .streamlit/secrets.toml file:
```
AWS_ACCESS_KEY_ID
AWS_SECRET_ACCESS_KEY
AWS_DEFAULT_REGION
OPENAI_API_KEY
BRIDGE_API_KEY
GOOGLE_MAPS_API_KEY
```

### How to run it on your own machine

1. Install the requirements

   ```
   $ pip install -r requirements.txt
   ```

2. Run the app

   ```
   $ streamlit run streamlit_app.py
   ```
